# VervetBiRC
Code developed for Bioinformatics Research Centre, Aarhus University 

## To run the code for creating vcf and bed file ##

```python
python src/create_bedvcf_file.py -in path/to/input -bed path/to/bed_output -vcf path/to/vcf_output
```


## To run the code changing vcf header ##

```python
python src/change_vcf_headers.py -in path/to/input.vcf -out path/to/output.vcf
```

## To run the code for calculating probablity of zero in all slices ##

```python
python src/probability_of_zero.py -in path/to/input -out path/to/out
```
